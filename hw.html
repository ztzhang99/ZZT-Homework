Explaination

  1. Method of complements:

    In mathematics and computing, the method of complements is a technique used to subtract one number from another using only addition of positive numbers.
    The method of complements is especially useful in binary (radix 2) since the ones' complement is very easily obtained by inverting each bit (changing '0' to '1' and vice versa). Adding 1 to get the two's complement can be done by simulating a carry into the least significant bit.

    The radix complement of an n digit number y in radix b is, by definition, bn-y. The radix complement is most easily obtained by adding 1 to the diminished radix complement, (bn-1)-y. Since bn-1 is the digit b-1 repeated n times (because bn-1=(b-1)(bn-1+bn-2+...+b+1)=(b-1)bn-1+...+(b-1)), the diminished radix complement of a number is found by complementing each digit with respect to b-1 (that is, subtracting each digit in y from b-1.

    在计算机科学里，我们无法直接定义radix complement为bn-y，而要通过diminished radix complement +1 来间接定义。为什么？假如我们要用b进制的n位表示一个整数，n位实际上是无法表示bn的。因此就要借助diminished radix complement。

  1. 请证明：二进制的负数（two's complement of X）等于 X 的 ones's complement ＋ 1（即，X每位求反加1）

    Answer:
    1. For n digits, -X = 2n - X
    2. One's complement of X equals to subtracting each digit in X from 1.
      Meanwhile, 11111...111(n digits) = 1*(2n-1+2n-2+...+2+1) = 2n -1.
      That means, one's complement of X equals to (2n -1) - X.
    3. So, two's complement of X equals to one's complement of X.

  2. Int8_t x = - 017; 请用8进制描述变量 x。

    Answer:
    1. Seven's complement of 017 equals to 060.
    2. Eight's complement of 017 equals to 060 + 1, which equals to 061.
    3. So, x can be represented by 061.

  3. C程序：int8_t x = -0x1f; int y = x; 请用16进制描述变量 x 和 y，并说明 int y = x 的计算过程。

    Answer:
    1. x : 111000012 = 0xE1
    2. y : 111111111111111111111111111000012 = 0xFFFFFFE1
    3. Proof:
      • First let's see what will happen if we add a digit-1 to the left of a subtractive number.
      • There is a subtractive number x. We add a digit-1 to the left of it, and we name the new number as y.
      • Suppose that the new digit-1 is the nth digit of y.
      • As we know, two's complement of x equals to 2n-1 - x, and two's complement of y equals to 2n - y.
      • For y = x + 2n-1, two's complement of y equals to 2n - x - 2n-1, which is also equals to 2n-1 - x.
      • So, two's complement of x equals to two's complement of y, which means, - x = - y.
      • Umm... Now we can conclude that x = y.
